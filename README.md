# 概要
利用者の位置情報を取得し、その位置の天気予報を表示するアプリ

# 使用技術
- React 17.0.2
- TypeScript 4.1.5
- ESLint
- Prettier

## 使用API
- Open Weather Map
- Geolocation API
- HeartRails Geo API

# 機能一覧
- 現在地の取得
- 取得した現在地をもとに天気を表示する
